mindsponge.function.get_ndarray
===============================

.. py:class:: mindsponge.function.get_ndarray(value, dtype=None)

    获取输入的ndarray类型。

    参数：
        - **value** (Union[int, Tensor, Parameter, ndarray]) - 输入的值。
        - **dtype** (type) - 数据类型。默认值：None。

    输出：
        ndarray。array类型的输入。
