mindsponge.common.get_aligned_seq
=================================

.. py:class:: mindsponge.common.get_aligned_seq(gt_seq, pr_seq)

    对比两个蛋白质序列。

    参数：
        - **gt_seq** (str) - 一条蛋白质序列。
        - **pr_seq** (str) - 另一条蛋白质序列。

    输出：
        str。一条蛋白质序列。
        str。两条序列的不同点。
        str。另一条蛋白质序列。
